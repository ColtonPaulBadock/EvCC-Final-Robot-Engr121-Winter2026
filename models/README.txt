This folder has all the models for 3D printing and for the robot!

Chassis-VersionOne:
	Our first take at making a prototype of how we want the chassis to look. We have mounting holes
	similar to a bread board so we can bolt down anything we want into the chassis (1/4 in holes).
	We included tread mounting extrustions (no lock in, they need to be glued) and walls we want
	to prototype for routing shafts outside the robot for running the treads and encoders.
